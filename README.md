## Keymap:
- LMB drag - draw rectangle (snaps to grid)
- RMB click - delete rectangle under cursor
- MMB drag - pan canvas
- Mouse‑wheel - zoom in/out
- "–" and "+" or  "[" and "]" - halve or double grid (like in Hammer Editor)
